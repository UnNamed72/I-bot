# I-bot
I-bot은 초보개발자 UnNamed72가 만든 카카오톡봇 입니다.

JavaScript로 작성되었고, 보라색 앱(젤브봇)이랑 파란색 앱(메신저 자동응답 봇)을 사용합니다.
